About This Demo

This demo showcases how you can enable secure Single Sign-On (SSO) for your applications using any SAML 2.0 compliant Identity Provider (IdP) with miniOrange. Our solution is compatible with all major IdPs, including but not limited to Azure AD, Okta, ADFS, Google, OneLogin, Ping, and more.

  • Easy integration with any SAML 2.0 IdP
  • Secure token handling and user session management
  • Seamless login experience for your users
  • Comprehensive support and documentation
